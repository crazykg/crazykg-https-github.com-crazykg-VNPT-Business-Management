# UI Rule Redirect

Tài liệu UI rule đầy đủ đã được sync vào:

- `AGENTS.md`

Quy ước hiện tại:

- `AGENTS.md` là entrypoint duy nhất cho workflow của repo.
- Khi task là UI, đọc section `UI Rule (Embedded)` trong `AGENTS.md`.
- Không dùng file này làm source of truth nữa.
- Nếu cần cập nhật rule UI, cập nhật trực tiếp trong `AGENTS.md` để giữ một nguồn duy nhất.
