## 6. Tổng hợp chức năng

### 6.1. CRUD Dự án
| Chức năng | Mô tả |
|-----------|-------|
| Tạo dự án | Modal form với mã, tên, mô tả, ngày, trạng thái |
| Sửa dự án | Mở lại modal với dữ liệu hiện tại |
| Xoá dự án | Soft delete, cần xác nhận (cascade toàn bộ UC con) |
| Nhân bản dự án | Tạo bản sao toàn bộ structure + UC |
| Danh sách | Bảng phân trang, tìm kiếm, lọc trạng thái |

### 6.2. CRUD Use Case (trong dự án)
| Chức năng | Mô tả |
|-----------|-------|
| Thêm UC | Modal 4 tabs: Thông tin, Các bước, Văn bản, Đính kèm |
| Sửa UC | Mở lại modal với đầy đủ dữ liệu |
| Xoá UC | Soft delete, cần xác nhận |
| Import Excel | Upload file → preview → cấu hình → import |
| Export Excel/PDF | Xuất danh sách UC theo filter hiện tại |
| Lọc theo nhóm | Sidebar cây phân cấp Subsystem → Group |
| Tìm kiếm | Fulltext theo tên, mã, mô tả |
| Đổi trạng thái | Dropdown: Draft → Reviewed → Approved → Implemented → Tested |

### 6.3. Quản lý Actor
| Chức năng | Mô tả |
|-----------|-------|
| Thêm/Sửa/Xoá Actor | CRUD đơn giản |
| Gán Actor vào UC | Multi-select trong form UC, phân biệt chính/phụ |

### 6.4. Quản lý phân cấp
| Chức năng | Mô tả |
|-----------|-------|
| Thêm/Sửa/Xoá Subsystem | CRUD trong sidebar |
| Thêm/Sửa/Xoá Function Group | CRUD trong sidebar, thuộc 1 Subsystem |
| Kéo thả UC giữa nhóm | Drag-and-drop thay đổi function_group_id |

### 6.5. Báo cáo
| Chức năng | Mô tả |
|-----------|-------|
| Thống kê tổng quan | Đếm UC, transaction, mức độ |
| Biểu đồ phân bổ | Bar chart theo nhóm chức năng |
| Tiến độ triển khai | Stacked bar theo trạng thái |
| Export báo cáo | PDF / Excel |

## 7. Responsive & UX

| Yêu cầu | Chi tiết |
|----------|----------|
| Desktop (>= 1024px) | Layout 2 cột: sidebar + content |
| Tablet (768-1024px) | Sidebar thu gọn (hamburger menu) |
| Mobile (< 768px) | Full-width, sidebar ẩn |
| Keyboard shortcuts | `Ctrl+N` = Thêm UC, `Ctrl+S` = Lưu, `Esc` = Đóng modal |
| Drag & drop | Sắp xếp bước UC, kéo UC giữa nhóm |
| Auto-save | Lưu nháp tự động mỗi 30s khi đang soạn văn bản |
| Toast notification | Thông báo thành công/lỗi ở góc phải trên |
